Инструкция установки

Infomir

  • 1. Находясь во внутреннем портале приставки выберите пункт «Настройки», далее «Системные настройки».
  • 2. Далее выберите пункт «Серверы».
  • 3. Далее откройте раздел «Порталы».
  • 4. В строку «Портал 1 URL» необходимо вписать адрес портала: stb.dinbox.tv
  • 5. Для сохранения адреса нажмите кнопку «ОК», затем - кнопку «EXIT» для возвращения в раздел «Системные настройки». Далее выбираем пункт «Перезапуск портала».

TVIP

  • 1. Выберите пункт "Настройки" и нажмите "ОК" на пульте.
  • 2. Далее в меню слева выберите пункт "ТВ" и перейдите в правую область стрелкой пульта "Вправо". В этом меню необходимо в пункте "Источник контента" стрелками выбрать вариант "Веб-портал", затем нажать на кнопку "Настроить Веб-портал"
  • 3. В появившемся окне установите пункт "Тип API" в значение "TVIP", а в поле "URL портала" введите адрес stb.dinbox.tv
  • 4. После заполнения URL нажмите кнопку "Применить", затем перезагрузите приставку. Для начала просмотра в главном меню выберите пункт "Телевидение".
